# Voice Chatbot Agent for Ride-Hailing & Food Delivery (Backend Sandbox)

Welcome to the **Voice Chatbot Agent** project, a specialized backend backend sandbox designed for simulating and validating voice-based customer support interactions in Ride-Hailing and Food Delivery domains.

This codebase integrates **ASR (Speech-to-Text) simulation**, **Intent & Entity Extraction (Regex-based Rule Engines)**, **Tool Calling Integration (REST endpoints matching OpenAPI/Gemini schema formats)**, and a **Mock Database Backend** with full state isolation to support automated testing.

---

## 🛠️ Setup & Installation

This project utilizes the ultra-fast Python package manager **`uv`**. Follow these instructions to setup your local environment:

### Prerequisites
Make sure Python 3.10+ is installed on your computer.

### 1. Synchronize Dependencies
Sync the virtual environment and install all packages locked in `uv.lock` by running:
```bash
uv sync
```
This command automatically creates a `.venv` virtual environment in your project directory and installs all required development and core libraries.

### 2. Generate Audio Samples
To generate the 20 `.mp3` test samples under `tests/audio_samples/` using Microsoft's Multilingual Neural Voices (Ava and Andrew), run:
```bash
uv run python generate_audio.py
```

## 🚀 Running the Server Locally

To launch the FastAPI development server, run:
```bash
uv run uvicorn app.main:app --reload
```
Once started, you can explore the interactive API docs at:
- Swagger UI: [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)
- ReDoc: [http://127.0.0.1:8000/redoc](http://127.0.0.1:8000/redoc)

## 🧪 Testing & Verification

We provide both automated unit tests (using Pytest) and a local integration pipeline script.

### 1. Automated Test Suite (Pytest)
Run the entire suite of 25 unit and integration tests. The database writes are isolated during tests to ensure zero seed data corruption:
```bash
uv run pytest -v
```

### 2. Local Static Flow Verification (Console Log Output)
To test the pipeline end-to-end (Audio Upload -> ASR -> Agent Decision -> Tool Execute -> Vietnamese Agent Response) with safe ASCII logs on Windows command line consoles:
```bash
uv run python test_static_flow.py
```
